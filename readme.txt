== Installation ==
This section describes how to install the "AI Title Generator" plugin and get it working.
e.g.

1. Go to `Plugins` in the Admin menu
2. Click on the button `Add new`
3. Search for `AI Title Generator` and click `Install Now`
4. Click on the `Activate plugin`
5. Enter Valid API Key to run "AI Title Generator"

You can also do it Manually.

1. Upload  `plugin-directory` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

Note : Before activating/installing this plugin, please ensure that you have installed and activated the WooCommerce plugin for generate product title.


== How to Generate OpenAI API Key ==
1.Go to OpenAI's Platform website at https://platform.openai.com and sign in with an OpenAI account.
2.Click your profile icon at the top-right corner of the page and select "View API Keys."
3.Click "Create New Secret Key" to generate a new API key.
